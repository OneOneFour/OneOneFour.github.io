Recap - key points from Lecture 1

  • Macrostate- Set of variables which describe the state of the whole system.
  • Microstate - Set of variables which describe the state of every individual particle in the system. Can also be thought of as the quantum eigenstate for the entire system.
  • Statistical Weight () - The number of microstates for a given microstate. Statistical weight is roughly equal to where is the number of particles in the system and is the energy of the system.
  • Single-Particle State - The set of microscopic variables for a single particle in a microstate. Not the same thing as a microstate.
  • Indistinguishable vs distinguishable - In most situations particles are indistinguishable, except when dealing with atoms in a solid.
  • Ensemble - The probability distribution for microstates in a system. Contains a collection of identical systems.
